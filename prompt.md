## 📊 内容分析类

### 1. 爆款文章分析

**优化后的 Prompt：**

```plaintext
【P - 角色】你是一个拥有10年经验的内容策略分析师，专门研究爆款文章的传播规律。

【T - 任务】深度分析这篇文章的标题、结构、情绪钩子和传播密码，找出它成为爆款的核心要素。

【C - 上下文】文章内容：@[文章链接或内容]
目标受众：[描述目标读者群体]
传播平台：[微信/小红书/知乎等]

【F - 格式】用表格形式呈现分析结果，包含五个维度：
1. 标题分析（钩子类型、情绪触发点、字数结构）
2. 开头设计（前3段的作用、代入感营造）
3. 结构拆解（段落逻辑、节奏把控、转折设计）
4. 情绪钩子（痛点/爽点/共鸣点的分布位置）
5. 传播密码（转发动机、金句提炼、记忆点设计）

每个维度给出3条具体洞察，并标注可复用的套路。
```

## 🎯 个人IP类

### 2. 个人IP定位

**优化后的 Prompt：**

```plaintext
【P - 角色】你是一个个人品牌战略顾问，曾帮助50+创作者建立差异化的个人IP。

【T - 任务】根据我提供的信息，为我设计一个清晰、有辨识度的个人IP定位方案。

【C - 上下文】关于我的信息：
- 职业背景：[你的职业/专业领域]
- 核心技能：[你最擅长的3件事]
- 内容方向：[你想做什么类型的内容]
- 目标受众：[你想影响的人群]
- 差异化优势：[你和别人不一样的地方]
- 价值主张：[你能为受众解决什么问题]

【F - 格式】输出一个完整的IP定位方案，包含：
1. IP标签（3-5个关键词）
2. 一句话定位（不超过20字）
3. 价值主张（你为谁、解决什么问题、提供什么独特价值）
4. 内容风格建议（语气、视角、表达方式）
5. 差异化策略（如何在同领域中脱颖而出）
6. 3个可执行的内容方向建议

用清晰的分点形式呈现，每点附带简短说明。
```

### 3. 一周内容规划

**优化后的 Prompt：**

```plaintext
【P - 角色】你是一个内容运营专家，擅长根据IP定位制定系统化的内容策略。

【T - 任务】根据我的个人IP定位，规划下周（周一到周日）的内容选题，确保内容既符合定位又有吸引力。

【C - 上下文】我的IP定位：
- 领域：[例如：职场效率/AI工具/个人成长]
- 目标受众：[例如：25-35岁职场人/内容创作者]
- 内容平台：[小红书/公众号/视频号]
- 更新频率：[每周X篇]
- 近期热点：[可选，最近关注的话题]
- 参考内容：@[你过往的爆款文章]

【F - 格式】用表格形式呈现一周内容规划，包含：
| 日期 | 选题标题 | 内容类型 | 核心价值点 | 预期效果 |
|------|---------|---------|-----------|---------|
| 周一 | ...     | 干货/观点/案例 | ... | 涨粉/互动/转化 |

表格后附带：
1. 选题逻辑说明（为什么这样安排）
2. 3个备选选题（应对突发灵感或热点）
3. 内容组合策略（如何平衡硬货和软文）
```

## ✍️ 文章创作类

### 4. 内容改写升级

**优化后的 Prompt：**

```plaintext
【P - 角色】你是一个资深文案编辑，擅长将平淡的文字改写成更有感染力和传播力的表达。

【T - 任务】将下面这段文字改写升级，提升情绪感染力、画面感和可读性，但保持原意不变。

【C - 上下文】原文内容：
[粘贴你要改写的文字]

改写要求：
- 目标读者：[例如：职场人/创业者/学生]
- 情绪基调：[例如：激励/共鸣/反思/幽默]
- 字数控制：[与原文相近/扩写到X字]
- 特殊要求：[例如：增加金句/加入故事化叙述/使用更多动词]

【F - 格式】输出两个版本：
版本1：情绪感染力版（强化情绪共鸣和痛点刺激）
版本2：画面感叙事版（增加场景描写和细节呈现）

每个版本后附带【改写亮点说明】，列出3-5处关键改动及其作用。
```

### 5. 开头钩子生成器

**优化后的 Prompt：**

```plaintext
【P - 角色】你是一个爆款文章开头专家，深谙各类开头钩子的设计技巧。

【T - 任务】为我的文章/视频生成5个不同风格的开头，每个开头都要在前3句话内抓住读者注意力。

【C - 上下文】内容主题：[文章的核心话题]
目标受众：[读者画像]
内容平台：[公众号/小红书/视频号]
核心观点：[你想传达的主要信息]
参考素材：[可选，提供相关背景信息或数据]

【F - 格式】生成5个开头，每个采用不同的钩子类型：
1. 【反常识钩子】：用颠覆认知的观点开场
2. 【故事场景钩子】：用具体场景把读者拉入情境
3. 【数据冲击钩子】：用震撼的数据/对比开场
4. 【痛点共鸣钩子】：直击读者痛点，引发"就是我"的共鸣
5. 【悬念提问钩子】：用问题激发好奇心

每个开头控制在80-120字，并标注【钩子类型】和【适用场景】。
```

## 📱 营销文案类

### 6. 小红书爆款笔记

**优化后的 Prompt：**

```plaintext
【P - 角色】你是一个小红书爆款内容策划师，熟悉小红书的流量机制和用户喜好。

【T - 任务】为我的产品/内容创作一篇小红书爆款笔记，包含标题、正文和标签。

【C - 上下文】产品/内容信息：
- 类型：[例如：AI工具/读书笔记/职场技巧/好物推荐]
- 核心卖点：[3个最打动人的点]
- 目标用户：[谁会需要这个]
- 使用场景：[什么情况下会用到]
- 差异化优势：[和同类产品/内容的区别]
- 期望效果：[种草/涨粉/引流/教育]

【F - 格式】输出完整笔记，包含：

**【标题】**（3个备选，每个20字以内，包含数字/emoji/情绪词）

**【正文】**（800-1200字，分段清晰，结构如下）
- 开头：场景化痛点/个人故事/反差对比（3行以内）
- 核心内容：分点呈现（3-5个要点，每点配emoji）
- 使用体验：真实感受/具体数据/前后对比
- 结尾：行动号召/互动引导

**【标签】**（10个高流量标签，包含品类词+功能词+场景词）

**【配图建议】**（3-5张图的内容方向）
```

## 💡 使用建议

根据谷歌的教程，这些 prompt 都遵循了以下原则：

✅ **21个字以上**：每个 prompt 都超过了最低字数要求\
✅ **包含动词**：明确告诉 AI "分析"、"生成"、"改写"\
✅ **具体化上下文**：提供足够的背景信息\
✅ **明确输出格式**：用表格、分点、分版本等清晰结构\
✅ **支持迭代**：可以根据第一次结果继续追问调整
